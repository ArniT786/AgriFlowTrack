# AgriFlowTrack — Backend Requirements
# This project uses native PHP (no Composer/external packages required).
# Listed below are the environment and server requirements needed to run the backend.

# ----------------------------
# Runtime
# ----------------------------
PHP >= 8.0
MySQL >= 5.7  (or MariaDB >= 10.3)

# ----------------------------
# Required PHP Extensions
# ----------------------------
pdo
pdo_mysql
json

# ----------------------------
# Local Server Environment
# ----------------------------
# Any one of the following is sufficient to run the backend locally:
XAMPP >= 8.0
# or
WAMP >= 8.0
# or
PHP built-in server (php -S localhost:8000)

# ----------------------------
# Database
# ----------------------------
# Database name: AgriFlow
# Connection configured in: db.php
# Default local credentials (update as needed):
#   host: localhost
#   username: root
#   password: (empty)

# ----------------------------
# Notes
# ----------------------------
# - No Composer dependencies are currently used in this project.
# - If external PHP packages are added in the future (e.g. PHPMailer, JWT auth),
#   they should be managed via composer.json instead of this file.

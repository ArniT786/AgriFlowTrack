# User Stories

## Farmer Stories

### US-01

As a Farmer, I want to register crop information so that I can track production activities.

### US-02

As a Farmer, I want to record harvest information so that I can maintain accurate production records.

### US-03

As a Farmer, I want to view inventory status so that I can monitor available stock.

---

## Processor Stories

### US-04

As a Processor, I want to view harvested crop details so that I can prepare packaging plans.

### US-05

As a Processor, I want to create product batches so that I can improve traceability.

### US-06

As a Processor, I want to record packaging information so that product history can be maintained.

---

## Warehouse Manager Stories

### US-07

As a Warehouse Manager, I want to monitor product shelf life so that I can reduce spoilage.

### US-08

As a Warehouse Manager, I want to receive expiration alerts so that I can prioritize product distribution.

### US-09

As a Warehouse Manager, I want to monitor warehouse inventory so that I can maintain stock accuracy.

### US-10

As a Warehouse Manager, I want FEFO recommendations so that products nearing expiration are distributed first.

---

## Administrator Stories

### US-11

As an Administrator, I want to manage users so that system access remains secure.

### US-12

As an Administrator, I want to generate reports so that operational performance can be analyzed.

### US-13

As an Administrator, I want to monitor system activity so that security risks can be identified.

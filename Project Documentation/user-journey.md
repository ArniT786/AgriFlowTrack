# User Journey

## Farmer Journey

### Stage 1: Login

User logs into AgriFlowTrack.

### Stage 2: Crop Registration

Farmer enters:

* Crop name
* Seed quantity
* Sowing date
* Farm information

### Stage 3: Harvest Recording

Farmer records:

* Harvest date
* Harvest quantity
* Crop condition

### Stage 4: Inventory Submission

Product inventory is updated and transferred to warehouse records.

### Outcome

Accurate production and inventory tracking.

---

## Processor Journey

### Stage 1: Login

Processor accesses the dashboard.

### Stage 2: View Harvest Information

Processor reviews:

* Crop details
* Harvest dates
* Quantity information

### Stage 3: Packaging

Processor records:

* Packaging date
* Package type
* Batch number

### Stage 4: Inventory Update

Packaged products are added to inventory.

### Outcome

Improved product traceability.

---

## Warehouse Manager Journey

### Stage 1: Login

Manager accesses warehouse dashboard.

### Stage 2: Inventory Monitoring

Review:

* Product quantity
* Storage location
* Batch information

### Stage 3: Shelf-Life Monitoring

Monitor:

* Expiration dates
* FEFO recommendations

### Stage 4: Storage Management

Update:

* Temperature
* Warehouse status

### Outcome

Reduced spoilage and better storage utilization.

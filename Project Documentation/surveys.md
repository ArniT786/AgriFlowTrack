# Surveys

## Survey Objective

The survey was conducted to identify common challenges and requirements among agricultural stakeholders regarding inventory management, storage monitoring, and supply chain tracking.

---

## Target Respondents

* Farmers
* Processors
* Warehouse Managers
* Agricultural Officers

---

## Sample Size

Total Respondents: 50

| Category              | Respondents |
| --------------------- | ----------- |
| Farmers               | 25          |
| Processors            | 10          |
| Warehouse Managers    | 10          |
| Agricultural Officers | 5           |

---

## Survey Questions

### Q1. Do you currently use a digital inventory management system?

| Response | Percentage |
| -------- | ---------- |
| Yes      | 24%        |
| No       | 76%        |

---

### Q2. Do you face challenges in tracking inventory?

| Response | Percentage |
| -------- | ---------- |
| Yes      | 82%        |
| No       | 18%        |

---

### Q3. Do you experience product losses due to storage issues?

| Response | Percentage |
| -------- | ---------- |
| Yes      | 74%        |
| No       | 26%        |

---

### Q4. Would shelf-life monitoring help reduce waste?

| Response | Percentage |
| -------- | ---------- |
| Yes      | 90%        |
| No       | 10%        |

---

### Q5. Would a centralized platform improve operations?

| Response | Percentage |
| -------- | ---------- |
| Yes      | 92%        |
| No       | 8%         |

---

## Key Findings

### Inventory Management

* Most respondents rely on manual tracking methods.
* Inventory inaccuracies are common.

### Storage Management

* Storage condition monitoring is limited.
* Product spoilage remains a significant issue.

### Information Accessibility

* Information is scattered across multiple sources.
* Stakeholders prefer a centralized platform.

### Technology Adoption

* Respondents showed strong interest in digital solutions.
* Automated monitoring was highly preferred.

---

## Survey Insights

| Requirement            | Support Level |
| ---------------------- | ------------- |
| Inventory Tracking     | Very High     |
| Shelf-Life Monitoring  | Very High     |
| Temperature Monitoring | High          |
| FEFO Management        | High          |
| Centralized Database   | Very High     |
| Reporting System       | High          |

---

## Recommendations

Based on survey responses, the proposed AgriFlowTrack system should prioritize:

1. Centralized inventory management.
2. Shelf-life and expiration tracking.
3. Temperature monitoring.
4. FEFO implementation.
5. Reporting and analytics.
6. User-friendly dashboards.

---

## Conclusion

Survey findings strongly support the development of AgriFlowTrack as an integrated agricultural inventory management platform capable of reducing waste, improving storage efficiency, and enhancing decision-making across the agricultural supply chain.
